About State Finder

 

The State Finder web application was created in the Fall of 2022 as a Senior Capstone project by Alyiah Proctor, Anne Nguyen, Darlyn Mendez, and Emily Crabtree, four students at the University of North Carolina at Greensboro. The goal of the web application is to be a tool to aid people undergoing a career transition, job search, and/or location change, and to make those processes less stressful. The State Finder web application collects user data based on their occupation, job level, number of working adults, and number of children in the household (up to 3). According to the occupation and job level provided, the user's likely salary (as collected by the United States Burea of Labor Statistics1) is found, and is then compared with the minimum livable wages of the major metropolitan areas in the United States of America for the number of working adults and children living in the household, as provided by MIT2.

By using the data provided by MIT, this application will give users a more accurate measure of where they can live comfortably, rather than whether or not they can afford a basic grocery budget. The living wage data takes into account a basic grocery budget for the household size, along with the costs of rent, utilities, child care (if all adults in the home are working), health insurance, transporation, and other rudimentary essentials, such as clothing and personal hygeine products. This calculated living wage surpasses wages below the poverty line, though does not account for nonessentials, such as traveling or entertainment.

From the data collected from the user, a red-green choropleth map is produced to visually illustrate which metropolitan areas for which the user meets the minimum livable wage, along with a button that redirects the user to a state infographic map. This map features a hover option for the user to learn more information about specific states, such as state capital, total population, number of housing units, the number and percentage of occupied housing units, number and percentage of vacant housing units, and a major attraction, as provided by the United States Census Bureau3. From there, the user can select a state from another menu, which provides a map of the selected state and the metropolitan areas within. By hovering over the areas on this map, the user can find population and housing information specific to the area.
