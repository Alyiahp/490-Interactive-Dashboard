About State Finder

 

The State Finder web application was created in the Fall of 2022 as a Senior Capstone project by Alyiah Proctor, Anne Nguyen, Darlyn Mendez, and Emily Crabtree, four students at the University of North Carolina at Greensboro. The goal of the web application is to be a tool to aid people undergoing a career transition, job search, and/or location change, and to make those processes less stressful. The State Finder web application collects user data based on their occupation, job level, number of working adults, and number of children in the household, and then compares it with the minimum livable wages of the major metropolitan areas in the United States of America, as provided by MIT. By using the data provided by MIT, this application will give users a more accurate measure of where they can live comfortably, rather than whether or not they can afford a basic grocery budget. The living wage data takes into account average grocery budget per household size, along with the costs of rent, utilities, child care (if all adultsin the home are working), health care, transporation, and other rudimentary essentials like clothing and personal hygeine products.

From the data collected from the user, a red-green choropleth map is produced to visually illustrate which Metropolitan areas the user meets the minimum livable wage, along with a menu of state names for the user to select for more information.
